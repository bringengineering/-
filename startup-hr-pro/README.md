- ✨ **Glassmorphism UI** - 현대적인 유리 효과 디자인

## 🚀 빠른 시작

### 1분 안에 실행하기

```bash
# 1. 프로젝트 폴더로 이동
cd startup-hr-pro

# 2. 브라우저에서 열기
open index.html  # Mac
start index.html # Windows
xdg-open index.html # Linux

# 또는 로컬 서버 실행 (권장)
python -m http.server 8000
# http://localhost:8000 접속
```

### 첫 화면
- 🎉 자동으로 샘플 데이터가 로드됩니다

- 🌙 우측 하단 버튼으로 다크모드 전환
- 📱 모바일에서는 좌측 상단 ☰ 버튼으로 메뉴 열기

### 1. 📊 통합 대시보드

## 🎨 디자인 시스템

### Glassmorphism
```css
background: rgba(255, 255, 255, 0.7);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.18);
```

### 애니메이션
- fade-in (0.5s)
- slide-up (0.5s)
- pulse-glow (2s infinite)
- shimmer (2s infinite)

## 🎓 실전 활용 가이드

### 신규 직원 입사
```
1. 직원 관리 → 직원 추가
2. 계약서 생성 → PDF 다운로드
3. 근태 기록 시작
```

### 52시간 초과 발견
```
1. 대시보드 긴급 알림 확인
2. 근태 관리 → 상세 확인
3. 즉시 퇴근 조치
4. 리포트 다운로드
```

### 직원과 통화 후
```
1. 대화 분석 → 내용 입력
2. 리스크 점수 확인
3. 보고서 다운로드 (증거)
```
