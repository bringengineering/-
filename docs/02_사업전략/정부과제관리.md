## 정부과제 관리 프로세스

### 연간 사이클

```
[1~2월] 공고 모니터링 → [2~3월] 사업계획서 작성 → [3~4월] 지원/선정
    → [수행기간] 연구개발 + 중간점검 → [종료] 최종보고 + 정산
```

### 관리 체크리스트

#### 지원 준비 단계
- [ ] 공고 내용 상세 분석
- [ ] 자격 요건 충족 여부 확인
- [ ] 사업계획서 초안 작성
- [ ] 기술 차별성 정리
- [ ] 예산 편성 (인건비, 재료비, 외주비 등)
- [ ] 대표 최종 검토 후 제출

#### 수행 단계 (매월)
- [ ] 연구노트 작성 (주 1회 이상)
- [ ] 지출 증빙 정리
- [ ] 마일스톤 진행률 체크
- [ ] 변경 사항 발생 시 변경 신청

#### 중간/최종 점검
- [ ] 보고서 작성 (2주 전 시작)
- [ ] 산출물 정리
- [ ] 증빙 서류 최종 점검
- [ ] 발표 자료 준비 (필요 시)
- [ ] 리허설 1회 이상

#### 정산 단계
- [ ] 지출 증빙 전체 취합
- [ ] 예산 대비 집행률 확인 (90% 이상 권장)
- [ ] 회계 장부 정리
- [ ] 정산 보고서 작성

## 예산 편성 기준 (정부과제 공통)

| 항목 | 비율 가이드 | 주의사항 |
|------|-----------|---------|
| 인건비 | 40~60% | 참여율, 4대보험 기준 |
| 직접비 (재료/장비) | 20~30% | 견적서 3사 비교 |
| 위탁연구비 | 10~20% | 사전 계약 필요 |
| 간접비 | 5~10% | 기관별 기준 상이 |

---

## 정부과제 정산 주의사항

1. **모든 지출은 증빙 필수**: 카드 영수증, 세금계산서, 이체 내역
2. **예산 항목 간 전용**: 20% 이내 자율, 초과 시 사전 승인
3. **인건비**: 4대보험 가입 증명, 출근부 관리
4. **장비 구매**: 50만원 이상은 자산 등록
5. **부가세**: 과세/면세 구분 정확히
