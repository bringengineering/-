## 성공 기준

12주 후 이것들이 **습관**으로 정착되었다면 성공:

| # | 체크 항목 | 상태 |
|---|----------|------|
| 1 | 데일리 스탠드업이 자연스럽게 매일 올라온다 | ☐ |
| 2 | 주간 킥오프/리뷰가 당연한 일과가 됐다 | ☐ |
| 3 | 월간 대시보드를 보고 경영 상황을 파악할 수 있다 | ☐ |
| 4 | OKR 기반으로 일하고, 체크인이 자연스럽다 | ☐ |
| 5 | 코드 리뷰/기술 싱크가 일상이다 | ☐ |
| 6 | 팀원 모두가 의사결정 권한을 알고 자율적으로 판단한다 | ☐ |
| 7 | 캐시플로우를 매월 점검하고 런웨이를 알고 있다 | ☐ |
