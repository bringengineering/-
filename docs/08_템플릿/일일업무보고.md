# 일일 업무 보고 (데일리 스탠드업)

## 작성 가이드
- **시간:** 매일 오전 10:00
- **형식:** 슬랙 #daily-standup 채널에 텍스트 작성
- **소요:** 5분 이내

---

## 양식

```
[데일리] {이름} / {YYYY-MM-DD}

✅ 어제:
- (완료한 일 1~2개)

📌 오늘:
- (할 일 1~2개, 가장 중요한 것 먼저)

🚨 이슈:
- (막힌 것, 도움 필요한 것. 없으면 "없음")
```

---

## 예시

```
[데일리] 노광민 / 2026-04-15

✅ 어제:
- VisiScan API 엔드포인트 설계 완료 (8개 엔드포인트)
- DB 스키마 초안 작성

📌 오늘:
- 측정 데이터 CRUD API 구현
- 배정환님과 AI 모델 인터페이스 협의

🚨 이슈:
- AWS RDS 인스턴스 생성 권한 필요 → 대표님 확인 요청
```

---

## 규칙
1. 길게 쓰지 않아도 됩니다. 3줄이면 충분
2. 이슈가 없으면 "없음"으로 적으면 됩니다
3. 하루 빠졌다고 스트레스 받지 마세요. 다음날 이어서
4. 이모지 없어도 됩니다 (선택사항)
