# 브링엔지니어링 — 예산 관리 기준

**최종 수정일:** 2026-03-10

---

## 예산 편성 원칙

1. **수입 보수적, 지출 현실적**: 수입은 확정된 것만, 지출은 여유분 포함
2. **정부과제 예산 기준 준수**: 항목별 비율, 전용 규정 엄격 준수
3. **예비비 10%**: 전체 예산의 10%는 예비비로 확보
4. **월간 점검**: 매월 예산 집행률 확인

---

## 정부과제 예산 항목 기준

| 비목 | 세목 | 비율 가이드 | 증빙 |
|------|------|-----------|------|
| **인건비** | 참여 연구원 인건비 | 40~60% | 급여대장, 4대보험 |
| **직접비** | 재료비 | 10~15% | 세금계산서, 견적서 |
| | 전산처리비 | 5~10% | 클라우드 이용료 |
| | 기술도입비 | 0~5% | 라이선스 계약서 |
| | 출장비 | 3~5% | 출장 보고서 |
| **위탁연구비** | 외부 연구 위탁 | 10~20% | 위탁 계약서, 결과물 |
| **간접비** | 관리/운영 경비 | 5~10% | 정률 적용 |

### 예산 전용 규칙
- **20% 이내**: 자율 전용 가능 (사전 기록)
- **20% 초과**: 전담기관 사전 승인 필요
- **인건비 → 다른 항목**: 원칙적 불가 (역방향은 가능)

---

## 지출 승인 기준

| 금액 | 승인자 | 필요 서류 |
|------|--------|----------|
| 10만원 미만 | 담당자 자율 | 영수증 |
| 10~50만원 | 대표 사전 승인 | 영수증 + 사유서 |
| 50만원 이상 | 대표 사전 승인 + 견적 비교 | 견적서 3사 + 승인서 |
| 100만원 이상 | 대표 사전 승인 + 견적 비교 | 견적서 3사 + 계약서 |

### 집행률 가이드
- **정상**: 월별 균등 집행 (예: 6월에 50% 내외)
- **주의**: 하반기에 몰아서 집행 → 부적격 의심
- **목표**: 연말 기준 **90~100%** 집행
