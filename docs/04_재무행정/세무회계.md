## 일상 회계 관리 규칙

### 지출 증빙 원칙
1. **모든 지출은 법인카드** 사용 (개인카드 사용 최소화)
2. **당일 증빙 처리**: 지출 발생 당일 영수증 사진 촬영 + 기록
3. **10만원 이상**: 세금계산서 발급 필수
4. **50만원 이상**: 사전 승인 후 지출

### 증빙 기록 양식
```
날짜: YYYY-MM-DD
금액: ₩
항목: [인건비/재료비/장비/출장/기타]
내용: [구체적 내용]
증빙: [영수증/세금계산서/이체내역]
정부과제 연관: [예/아니오, 과제명]
```

### 월간 회계 체크리스트
- [ ] 법인카드 사용 내역 정리
- [ ] 세금계산서 수취 확인
- [ ] 인건비 원천세 신고 (10일)
- [ ] 4대보험료 납부
- [ ] 캐시플로우 실적 업데이트
- [ ] 정부과제 예산 집행률 확인

## 핵심 재무제표 (분기별 확인)

### 1. 손익계산서 (P&L)
- 매출, 비용, 영업이익을 보여줌
- "돈을 벌고 있는가?"

### 2. 재무상태표 (B/S)
- 자산, 부채, 자본을 보여줌
- "회사의 재정 건강 상태는?"

### 3. 현금흐름표 (C/F)
- 현금의 유입/유출을 보여줌
- "실제로 현금이 있는가?"

> 스타트업에서 가장 중요한 것: **현금흐름표**
> 이익이 나도 현금이 없으면 망합니다 (흑자 도산)
