## IP 분류 체계

### 1. 특허 (Patent)
- 핵심 기술에 대한 독점적 권리
- 출원 → 심사 → 등록 과정 (1.5~2년)

### 2. 영업비밀 (Trade Secret)
- 특허 출원 없이 보호하는 기술/노하우
- 비밀 유지 계약(NDA)으로 보호

### 3. 저작권 (Copyright)
- 소프트웨어 소스코드, 문서, 디자인
- 자동 발생 (별도 등록 불필요, 등록 시 입증 용이)

### 4. 상표 (Trademark)
- 회사명, 제품명, 로고
- 브링엔지니어링, VisiScan, Shadow AI 등

## IP 관리 프로세스

### 신규 IP 발굴

| 시기 | 방법 |
|------|------|
| 격주 기술 싱크 | "이번에 개발한 것 중 특허 가능한 것?" 체크 |
| 분기 OKR 리뷰 | IP 성과 점검 |
| 논문/발표 전 | 공개 전 특허 출원 가능성 검토 (공개 후 출원 불가!) |

### 특허 출원 판단 기준

| 기준 | 질문 |
|------|------|
| 신규성 | 세상에 없는 새로운 것인가? |
| 진보성 | 기존 기술 대비 진보된 것인가? |
| 사업성 | 이 특허가 매출/경쟁력에 기여하는가? |
| 방어가치 | 경쟁사가 이 기술을 쓰면 우리가 불리한가? |

→ 4개 중 3개 이상 "Yes"면 출원 추진

## IP 보호 규칙

### 모든 팀원 필수 준수

1. **NDA 서명**: 입사 시 비밀유지 계약 서명
2. **소스코드 관리**: Private 레포지토리만 사용, 외부 공개 금지
3. **외부 발표 전 검토**: 기술 내용 포함된 발표/논문은 대표 사전 승인
4. **SNS 주의**: 기술 상세, 내부 전략을 SNS에 게시 금지
5. **퇴사 시**: IP 관련 자료 반납, 비밀유지 의무 지속 (2년)

### 정부과제 IP 주의사항

- 정부과제로 개발된 기술의 IP는 **수행기관(브링)에 귀속** (일반적)
- 단, 과제별 협약서 확인 필수
- 실시권, 보고 의무 등 조건 준수
